# 컴퓨터 네트워크

## 컴퓨터 네트워크란?

여러 개의 장치가 그물망처럼 연결되어 정보를 주고받을 수 있는 통신망을 **컴퓨터 네트워크**라고 합니다.


## 네트워크의 네트워크, 인터넷

인터넷은 **네트워크의 네트워크(Network of Networks)**,  
즉 여러 개의 네트워크를 서로 연결한 거대한 네트워크를 의미합니다.


## 개발자가 컴퓨터 네트워크를 알아야 하는 이유

개발 업무는 크게 **새로운 프로그램 개발**과 **프로그램 유지보수**로 나눌 수 있습니다.  
네트워크 지식은 이 두 영역 모두에서 중요한 역할을 합니다.

예를 들어,

- **TCP / UDP**를 모르면 스프링 프레임워크의 동작 원리를 이해하기 어렵습니다.
- **HTTP와 쿠키**를 모르면 로그인 기능을 구현하기 어렵습니다.

또한 실제 개발 환경에서는 다음과 같은 네트워크 관련 개념을 자주 접하게 됩니다.

- DNS  
- HTTP / HTTPS  
- 포트 번호  

프로그램 유지보수 과정에서도 네트워크 지식이 필요합니다.  
예를 들어 다음과 같은 명령어를 사용할 때도 기본적인 네트워크 이해가 필요합니다.

```
netstat -ano
```
